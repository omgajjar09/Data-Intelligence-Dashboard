# 📊 Excel Data Analysis & Dashboard Project

## 📁 Project Overview

This project focuses on **advanced Excel data analysis, automation, and dashboard creation** using a real-world transactional dataset. It demonstrates practical implementation of Excel functions, data analysis tools, and visualization techniques to generate meaningful business insights.

---

## 🧾 Dataset Description

The dataset contains transactional sales data with the following key fields:

* Transaction ID
* Date
* Customer ID & Name
* Product ID & Name
* Category
* Quantity & Unit Price
* Total Amount
* Payment Method
* Region
* Customer Segment
* Customer Since

---

## 🎯 Objectives

* Perform **data cleaning and transformation**
* Apply **advanced Excel formulas**
* Conduct **data analysis & statistical modeling**
* Build an **interactive dashboard**
* Generate **business insights & storytelling**

---

## 🛠️ Key Features Implemented

### ⏱️ Date & Time Functions

* `TODAY()` → Current date tracking
* `NOW()` → Timestamp creation
* `DATEDIF()` → Customer age analysis
* `EOMONTH()` → Monthly grouping

---

### 🔍 Data Filtering & Lookup

* `FILTER()` → Extract dynamic multi-value results
* Matching datasets using comparison formulas

---

### 🎨 Conditional Formatting

* KPI indicators using:

  * Icons (▲ ▼)
  * Color scales
* Monthly sales growth visualization

---

### ⏳ Timestamp System

* Created using `NOW()` with **iterative calculation**
* Used for tracking updates dynamically

---

### 📈 WHAT-IF Analysis

* Scenario Manager → Business simulation
* Goal Seek → Target-based calculations

---

### 📊 Statistical Analysis

* **Linear Regression** using Data Analysis Toolpak
* **Descriptive Statistics**:

  * Mean
  * Median
  * Standard Deviation
  * Variance

---

### 🧮 Customer Analysis

* Identified **High-Value Customers** using:

  * Aggregation formulas
  * Group-based calculations
* Total purchase per customer

---

### 🛒 Product Insights

* Most frequently purchased products identified
* Category-wise and product-wise sales analysis

---

### 🔗 Data Comparison

* Compared two lists and extracted:

  * Matching names
  * Duplicate entries

---

### ✂️ Text Functions

* Generated abbreviations using:

  * `LEFT()`, `RIGHT()`, `MID()`, `TEXT()`

---

## 📊 Dashboard Features

### 🔹 Pivot Tables

* Sales by Category
* Sales by Region
* Sub-category analysis

### 📉 Pivot Charts

* Bar Chart → Category comparison
* Line Chart → Sales trends
* Pie Chart → Contribution analysis

### 🎛️ Interactive Controls

* Slicers for:

  * Region

### 📌 KPI Indicators

* Total Sales
